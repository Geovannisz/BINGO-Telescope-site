# Security Policy

## Supported Versions

Currently, only the main branch of the BINGO Telescope Website is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| `main`  | :white_check_mark: |
| older   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project or the Decap CMS integration, please do not disclose it publicly. Instead, please report it via email to the project maintainers. 

We take all security vulnerabilities seriously and will work to address them promptly. You should receive a response within 48 hours. If the issue is confirmed, we will release a patch as soon as possible.
