# Contributing to BINGO Telescope Website

First off, thank you for considering contributing to the BINGO Telescope website! It's people like you that make this scientific project a great collaborative effort.

## How to Contribute

1. **Fork the repository** to your own GitHub account.
2. **Clone the project** to your local machine.
3. **Create a branch** for your feature or bug fix (`git checkout -b feature/my-new-feature`).
4. **Make your changes**. Ensure that your code follows our existing styling and formatting rules.
5. **Run tests**. Verify that your changes don't break existing functionality (`npm run check` and `npm run build`).
6. **Commit your changes** with clear and descriptive commit messages (`git commit -m "feat: add new feature"`).
7. **Push to your branch** (`git push origin feature/my-new-feature`).
8. **Open a Pull Request** against the `main` branch.

## Content Management

If you are updating team members, news, or publications, you do not need to edit the source code. You can use the integrated CMS available at `/admin/` (requires proper credentials).

## Issues

If you find a bug or have a suggestion for an improvement, please use the issue tracker to report it.

Thanks again for your support!
