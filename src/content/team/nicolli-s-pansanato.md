project_title: Pipeline de Alta Resolução para o BINGO
project_description: >-
  Esta pesquisa tenta obter dados com maior resolução para o radiotelescópio
  BINGO usando um algoritmo matemático e computacional. Na prática, isso
  permitiria gerar mapas do céu onde pequenas estruturas, antes “borradas”,
  pudessem ser detectadas com nitidez.


  Para entender melhor, imagine que você tire uma foto da Lua com um celular comum. Os detalhes da superfície - como crateras - saem borrados. Depois, você tira outra foto com um telescópio e uma câmera profissional: agora os detalhes ficam mais nítidos. Essa diferença é um aumento de resolução.


  Agora imagine que você consegue o mesmo aumento de resolução sem trocar o celular pelo telescópio, apenas deixando o celular comum olhando para o céu por mais tempo e, depois, aplicando um algoritmo na foto. É isso que minha pesquisa almeja fazer para o BINGO.
project_problem: Minha pesquisa busca contornar o limite deresolução que o
  radiotelescópio BINGO possui instrumentalmente.
project_importance: Isso é importante porque, posteriormente, possibilitaria a
  detecção de estruturas relevantes para o estudo da evolução do Universo e
  também ajudaria no processo de remoção de sinais contaminantes nos mapas
  obtidos.
project_challenges: Um dos principais desafios que precisei enfrentar foi
